# GST BillBook - Installation Guide

## Installation Steps:

1. Install Python 3.11 from: https://www.python.org/downloads/
   - ✅ Check "Add Python to PATH" during installation

2. Open Command Prompt and install dependencies:


3. Double-click `start_app.bat` to run the application

4. Your browser will open automatically at http://localhost:8501

## First Time Setup:
- Go to "Company" tab and enter your business details
- Upload your logo in the sidebar
- Add your UPI ID for QR codes
- Click "Save Settings"
